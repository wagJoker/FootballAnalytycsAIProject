# C4: Context

## System: Football YOLO Analytics

**Football YOLO Analytics** — система для детекции и трекинга футболистов на видео с использованием моделей YOLO (YOLOv8, YOLOv12).

### Actors

- **ML Engineer / Student** – обучает YOLO/YOLO12 модели на футбольных датасетах, проводит A/B тестирование различных конфигураций моделей.

- **Analyst / Coach** – использует результаты детекции и трекинга для анализа футбольных матчей и получения инсайтов.

### External Systems

- **Kaggle** – предоставляет датасет `adilshamim8/football-players-detection` с размеченными изображениями футболистов.

- **GitHub** – хостинг исходного кода и CI/CD пайплайнов.

- **Ultralytics** – библиотека для работы с YOLO моделями.

### System Inputs

- Видеофайлы футбольных матчей (MP4, AVI и др.)

- Размеченные изображения из Kaggle датасета (YOLO формат)

### System Outputs

- Обученные модели детекции (веса в формате `.pt`)

- CSV файлы с треками игроков (frame, track_id, координаты)

- Метрики A/B тестирования и визуализации результатов

- Аннотированные видео с детекциями и треками

### Key Features

- Автоматическое скачивание и подготовка датасета Kaggle

- Обучение нескольких конфигураций моделей (A/B тестирование)

- Трекинг игроков на видео с использованием ByteTrack

- Визуализация метрик и статистики трекинга
